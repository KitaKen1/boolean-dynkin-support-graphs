# Program Code

This directory contains the small scripts needed to rebuild the visible
GitHub-facing figures and smoke-test the package.

| file | role |
|---|---|
| `plot_dynkin_type_distribution_1_to_5.py` | draws the n=1..5 Dynkin type presence matrix. |
| `plot_dynkin_diagram_shapes.py` | draws the A/D/E Dynkin shape legend. |
| `plot_boolean_to_dynkin_pipeline.py` | draws the Q3 Boolean-to-support-graph explanatory SVG. |
| `check_f169_case_study.py` | checks the compact F169 case-study data. |
| `validate_package.py` | smoke-tests CSV/JSON parsing, SVG presence, and local links. |

The package keeps compact data tables and figure-generation code only.
